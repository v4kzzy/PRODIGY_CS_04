# File Integrity Monitor (Prodigy Task 04)

This tool is a Keylogger alternative that focuses on defensive cybersecurity. It calculates the SHA-256 hash of files to verify their integrity and detect unauthorized modifications.

## Features
- **SHA-256 Hashing:** Generates a unique fingerprint for any file.
- **Integrity Verification:** Compares current hashes against expected values to detect tampering.
- **GUI:** Built with `ttkbootstrap` for a modern, dark-mode interface.
